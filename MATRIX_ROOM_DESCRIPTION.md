Work Hours Bot — tracks your daily working times. Send `!help` to see all commands.

Quick reference:
- `!today` — start today's entry
- `!status` — check today's logged hours
- `!correct DD.MM 09:00 17:30 12:00-12:30` — fix a day's entry
- `!correct DD.MM k/u/g` — mark sick / vacation / flexday
- `!missed [DD.MM]` — see missing entries from the last 14 days (optional: check a specific date)
- `!missed month [MM]` — see all missing entries for a full month (default: current month)
- `!language de|en` — switch language
